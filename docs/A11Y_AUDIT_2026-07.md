**Modal.tsx 개선 사항 (이번 감사에서 수정):**
- `role="dialog"` + `aria-modal="true"` + `aria-labelledby` 추가
- ESC 키 `keydown` 핸들러 추가 (`useEffect`)
- 백드롭에 `aria-hidden="true"` 추가

**미해결 항목:**
- `Modal.tsx` 포커스 트랩: 커스텀 구현이므로 `focus-trap-react` 라이브러리 도입 또는 수동 구현 필요. 현재 ESC·백드롭 닫기는 작동함. 추후 작업으로 분류.
- `NotificationPanel.tsx` ESC 닫기: 바깥 클릭 닫기는 구현되어 있으나 ESC 이벤트 확인 필요.

## 4. PWA Shortcuts iOS 참고

`public/manifest.json`의 `shortcuts` 필드는 **Android(Chrome)** 및 **데스크톱 브라우저**에서 앱 아이콘 길게 누르기(컨텍스트 메뉴)로 동작합니다.  
**iOS Safari는 `shortcuts`를 지원하지 않습니다.** iOS에서 홈 화면 앱 아이콘을 길게 눌러도 단축키 목록이 표시되지 않습니다 (iOS 17 기준).
